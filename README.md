## 📊 Project Overview

</div>

<div align="center">

### 🎯 Empowering Smart Investment Decisions with AI

</div>

<p align="center">
  <strong>Dhan-Laabh</strong> is a comprehensive, end-to-end Machine Learning application that revolutionizes stock market analysis. By combining advanced time-series forecasting, real-time sentiment analysis, and sophisticated statistical models, we provide investors with accurate price predictions and actionable insights.
</p>

## 📂 Project Structure

</div>

<div align="center">

### 🏗️ **Well-Organized Codebase** 🏗️

</div>

```
📁 Dhan-Laabh/
│
├── 🚨 exception/              # Custom exception handling
├── 🌐 flask_app/              # Web application
│   ├── 🎯 artifacts/          # Model artifacts and saved models
│   ├── 💭 sentiment_analysis/ # Sentiment analysis components
│   ├── 🎨 static/             # CSS, JavaScript, and images
│   ├── 📄 templates/          # HTML templates
│   ├── ⚡ app.py              # Main Flask application
│   └── 📋 requirements.txt    # Flask app dependencies
│
├── 📝 logger/                 # Logging configuration
├── 🤖 stock_prediction/       # Core ML pipeline components
├── 📊 notebook.ipynb          # Development notebook with analysis
├── 🔄 dvc.yaml               # Data Version Control configuration
├── ⚙️ params.yaml            # Model parameters
├── 🛠️ setup.py               # Package setup configuration
└── 📖 README.md              # Project documentation
```

<div align="center">

### 📥 **Step-by-Step Setup Guide**

</div>

<table align="center">
<tr>
<td align="center" width="20%">

**1️⃣**
### Clone Repository

</td>
<td align="left" width="80%">

```bash
git clone https://github.com/tgarg535/Dhan-Laabh.git
cd Dhan-Laabh
```

</td>
</tr>
<tr>
<td align="center" width="20%">

**2️⃣**
### Setup Environment
*(Optional but Recommended)*

</td>
<td align="left" width="80%">

```bash
python -m venv venv

# On Windows
venv\Scripts\activate

# On macOS/Linux
source venv/bin/activate
```

</td>
</tr>
<tr>
<td align="center" width="20%">

**3️⃣**
### Install Dependencies

</td>
<td align="left" width="80%">

```bash
cd flask_app
pip install -r requirements.txt
```

</td>
</tr>
<tr>
<td align="center" width="20%">

**4️⃣**
### Launch Application

</td>
<td align="left" width="80%">

```bash
python app.py
```

</td>
</tr>
<tr>
<td align="center" width="20%">

**5️⃣**
### Access Application

</td>
<td align="left" width="80%">

Open your browser and navigate to:
**`http://localhost:5000`**

</td>
</tr>
</table>

<br>

🎉 **You're all set! Start predicting stock prices now!** 🎉
